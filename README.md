# Ikigai
## Technology Used

- Next.js
- Next-Auth
- Cloudinary
- Mongoose or MongoDB

## Getting Started

To get started with ikigai, you will need to have Node.js installed on your computer.

1. Clone the repository to your local machine.
2. Run `npm install` to install the project's dependencies.
3. Set up a MongoDB database and add the database URL to `.env` file.
4. Set up a Cloudinary account and add the Cloudinary details to `.env` file.
5. Run `npm run dev` to start the development server.
